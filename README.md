# CarsAnalytics.WebApp

A modern Blazor WebAssembly frontend for the CarsAnalytics platform, inspired by sites like **Edmunds.com**, providing interactive UI, data exploration, comparisons, reviews, ratings, and administrative tools.

This application consumes **CarsAnalytics.ExperienceApi** and **CarsAnalytics.SystemApi**, offering a clean, modular, and scalable UI built with **Blazor**, **MudBlazor**, and **Fluxor**.
